# Hostel Outing Monitoring System

This is a Web-Based Dashboard powered by a native C++ Backend. 

## How to Run This Project Any Time You Want

Whenever you close the project and want to run it again, simply follow these two steps:

**1. Open a Terminal and run this exact command:**
```bash
cd /Users/udaykumar/.gemini/antigravity/scratch/hostel_monitor && g++ -std=c++11 main.cpp -o main && python3 server.py
```
*(You can just copy and paste this command every single time).*

**2. Open your Web Browser and go to:**
http://localhost:8086

---

### Tips
- Leave the terminal open while you use the website.
- When you are completely done for the day, go to the terminal and press **Ctrl + C** to turn the server off safely.
